弱者道之用……

Garry Tan是一位自己白手起家年入八位数的姐姐推荐给我的，她特别喜欢 Garry 把一些有价值的技能比作” beyond API 的技能”。

  1. 要么学习（ learning ）要么赚钱（ earning )。如果你既没在学习又没在赚钱，那就是正在浪费生命
  2. 无论做创作者（ Maker ）还是推动者
    ( Hustler )，尽早开始学习，尽早成为某事的专家。要么成为创造者（ Maker ，涉及设计、产品和工程），要么成为推动者（ Hustler ，涉及管理、销售和市场营销）
  3. 关注技能（能创造财富的引擎和发动机的技能）而不是金钱本身，尤其是那些罕见的、少见的、没人知道怎么做的、或者组合起来很罕见的技能。
    Garry 用的词是” Beyond API “的技能，比如 Uber 的程序通过 APP 分发订单，开车就是 API 以下的技能，做一家Uber公司就是” Beyond API “的技能。
  4. 真的”拥有技能”而不是”看上去拥有了”技能。每天分析很多、学习了很多，但自己亲自上手啥也做不出来都是假技能。
  5. learning （学习）和 earning （赚钱）哪个先哪个后？整体上最好先 learning 再 earning 。 Tim Cook 说，什么都是不确定的： the only thing I believe is preparation
  6. 钱不是真正的财富，而是一种记账方式。金钱与真正的财富的关系，是商品和服务的关系，是单词与物理世界的关系。
  7. 谈论赚钱可能会使真正的赚钱变得更加困难，会陷入追求晋升和上升流动的困境。
  8. 真正的财富来自于技能、创新和解决他人的问题，将这个概念视为一个财富机器。”放钱进去，然后转动曲柄。那台机器会吐出比开始时更多的钱。”
  9. 钱不能带来幸福，因为幸福不依赖「象征权力和财富」的东西，而依赖于规律生活( Disciplined life )。 Alan Watts ：真正的幸福和满足感只与内在的纪律和价值观有关。
  10. 如果你年轻且节俭，请不要改变你的生活方式，这样时间可以被省下来做更重要的
